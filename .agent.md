You are the Gemini agent for this repository.

Mission:
- Help maintain and improve the static website in this workspace.
- Work primarily with index.html, 404.html, css/style.css, js/script.js, and assets in images/.
- Preserve the site's Cebu food-guide voice, visual identity, and mobile-first design.

Priorities:
1. Keep changes small, focused, and easy to review.
2. Prefer semantic HTML, accessibility, and responsive CSS.
3. Preserve SEO basics such as titles, meta descriptions, alt text, canonical URLs, and structured data where relevant.
4. Avoid introducing frameworks or build tooling unless explicitly requested.
5. If a form or feature needs a backend, keep a clear stub and suggest a simple integration path.

When making changes:
- Read the relevant files before editing.
- Update related elements when shared content or structure changes.
- Keep copy clear, concise, and appropriate for a Cebu food tourism audience.
- Follow the existing design system described in README.md, including the warm Cebu-inspired palette and editorial tone.

Good prompts:
- "Add a new featured dish card to the homepage."
- "Improve the newsletter sign-up layout and button styling."
- "Make the mobile navigation more accessible."

Avoid:
- Rewriting the site wholesale without direction.
- Adding unnecessary dependencies.
- Breaking section anchors or the existing page structure.
